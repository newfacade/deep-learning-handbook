LSTM¶

import torch
from torch import nn
import d2l

Model¶

LSTM shares many of the properties of the GRU.

Interestingly, LSTMs have a slightly more complex design than GRUs but predates GRUs by almost two decades.

LSTM have three gates: the input gate \(\mathbf{I}_{t}\in\mathbb{R}^{n\times{h}}\), the forget gate \(\mathbf{F}_{t}\in\mathbb{R}^{n\times{h}}\) and the output gate \(\mathbf{O}_{t}\in\mathbb{R}^{n\times{h}}\).

\[\mathbf{I}_{t} = \sigma(\mathbf{X}_{t}\mathbf{W}_{xi} + \mathbf{H}_{t-1}\mathbf{W}_{hi} + \mathbf{b}_{i})\]
\[\mathbf{F}_{t} = \sigma(\mathbf{X}_{t}\mathbf{W}_{xf} + \mathbf{H}_{t-1}\mathbf{W}_{hf} + \mathbf{b}_{f})\]
\[\mathbf{O}_{t} = \sigma(\mathbf{X}_{t}\mathbf{W}_{xo} + \mathbf{H}_{t-1}\mathbf{W}_{ho} + \mathbf{b}_{o})\]

Besides hidden state, LSTM introduces the memory cell \(\mathbf{C}_{t}\in\mathbb{R}^{n\times{h}}\), engineered to record additional information.

To update memory cell, candidate memory which performs like a gate except it uses \(\mbox{tanh}\) instead of \(\sigma\) is needed:

\[\tilde{\mathbf{C}}_{t} = \mbox{tanh}(\mathbf{X}_t\mathbf{W}_{xc} + \mathbf{H}_{t-1}\mathbf{W}_{hc} + \mathbf{b}_c)\]

Next the forget gate and the input gate come to play.

Forget gate \(\mathbf{F}_{t}\): how much of the old memory we retain.

Input gate \(\mathbf{I}_{t}\): how much of the new information we use.

\[\mathbf{C}_{t} = \mathbf{F}_{t}\odot\mathbf{C}_{t-1} + \mathbf{I}_{t}\odot\tilde{\mathbf{C}}_{t}\]

Finally, combine output gate and memory cell to generate hidden state:

\[\mathbf{H}_{t} = \mathbf{O}_{t}\odot\mbox{tanh}(\mathbf{C}_{t})\]

Train¶

batch_size, num_steps = 32, 40
train_iter, vocab = d2l.load_data_novels(batch_size, num_steps)

num_hiddens = 256
lstm_layer = nn.LSTM(len(vocab), num_hiddens)
net = d2l.RNNModel(lstm_layer, vocab_size=len(vocab))

num_epochs, lr = 50, 0.1
d2l.train_language_model(net, train_iter, vocab, lr, num_epochs)
